# Farm War Alliance Base Design Page

This is a replica of the Farm War Alliance base design page created based on https://farmwaralliance.org/base-design/

## Files Created

- `index.html` - The main HTML page with all content and styling

## Features

- Responsive design that works on mobile and desktop
- Clean, organized layout with color-coded sections
- Text on one side, image placeholders on the other side for each section
- Base instruction images showing what's allowed vs not allowed
- Direct image display (not just links)
- FWA Base Guide with links to base pictures and copy layout links
- Interactive elements (placeholder links for social media and Google Slideshow)
- Proper HTML5 structure with semantic elements
- CSS styling with modern design principles

## Content Sections

1. **Header** - Title and main branding
2. **Four Base Sections Overview** - Visual summary of the required sections
3. **General Requirements** - FWA base design rules
4. **Town Hall Section** - Text on left, image placeholder on right
5. **King Section** - Text on left, image placeholder on right
6. **Queen Section** - Text on left, image placeholder on right
7. **Traps Section** - Text on left, image placeholder on right
8. **Base Instruction Images** - Visual examples of allowed vs not allowed setups
9. **FWA Base Guide** - Complete guide with base pictures and copy layout links
10. **Intellectual Property Notice** - Legal disclaimer
11. **Follow Us** - Social media links (placeholder)
12. **Footer** - Copyright information

## Image Structure

- Each of the 4 main sections (Town Hall, King, Queen, Traps) has:
  - Text content on one side
  - Image placeholder on the other side
  - Descriptive captions for images
- Base instruction images show:
  - ✅ ALLOWED examples (green border)
  - ❌ NOT ALLOWED examples (red border)
  - Visual comparisons of correct vs incorrect setups
- Base pictures are linked to imgur.com for each TH level (TH9-TH18)

## Styling

- Green color scheme representing FWA branding
- Card-based layout for content sections
- Responsive design with media queries
- Hover effects and visual hierarchy
- Accessible color contrast

## Notes
